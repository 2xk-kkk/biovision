光合作用与呼吸作用
光合作用：绿色植物在叶绿体中利用光能将 CO₂ 和 H₂O 转化为有机物（如葡萄糖）并释放 O₂。
总反应式：6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂（需光、叶绿体）
有氧呼吸：细胞在线粒体中分解有机物释放能量，生成 CO₂ 和 H₂O。
总反应式：C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O + 能量
